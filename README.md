Para instalar todas las dependencias que utiliza el proyecto debe correr los siguientes comandos en un terminal:

### `cd src`

### `npm install` 

Para iniciar la aplicación, en el directorio del proyecto, usted puede correr los comandos en un terminal:

### `npm start`

Para correr la alicación en modo de desarrollo:
Poner en el navegador la url: http://localhost:3000
